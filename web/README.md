# H&Z website redesign (preview)

Interactive homepage preview for the HZCEng design spec. Uses **Vite**, **React**, **TypeScript**, and **Tailwind CSS** v4.

- Run from repo root: `npm run dev` (or `cd web` then `npm run dev`)
- Build: `npm run build`
- Production imagery: replace Unsplash placeholders with H&Z project photos

Brand tokens live in `src/index.css` (`@theme`).
